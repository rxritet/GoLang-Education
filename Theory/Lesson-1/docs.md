# День 1: Основы Go на примере `main.go`

## Что изучаем в первый день

Цели урока:

1. Понять минимальную структуру программы на Go.
2. Научиться выводить данные в консоль.
3. Научиться считывать ввод пользователя.
4. Освоить базовые операции с целыми числами: деление и остаток.

Разбор построен по файлу `Practice/Lesson-1/main.go`.

## Структура файла

В файле показаны три учебных этапа:

1. `Hello World` (закомментирован).
2. Сумма двух чисел (закомментирован).
3. Перевод градусов в часы и минуты (активный `main`).

Важно: в одном пакете может быть только **одна активная** функция `main`, поэтому предыдущие варианты закомментированы.

## Базовые правила Go, которые здесь встречаются

### `package main`

Обозначает исполняемую программу.

### `import "fmt"`

Подключает стандартный пакет ввода-вывода:

1. `fmt.Println(...)` выводит текст и значения.
2. `fmt.Scan(...)` считывает значения с клавиатуры.

### `func main()`

Точка входа программы. Выполнение начинается отсюда.

### Переменные

Примеры:

```go
var a int
var a, b int
```

Тип `int` хранит целые числа.

## Этап 1: Hello World

Пример из файла:

```go
// func main() {
// 	fmt.Println("Hello, World!")
// }
```

Что тренируем:

1. Синтаксис функции `main`.
2. Вывод текста в консоль.
3. Первый успешный запуск программы.

## Этап 2: Сумма двух чисел

Пример из файла:

```go
// func main() {
// 	var a, b int
// 	fmt.Scan(&a, &b)
// 	fmt.Println(a + b)
// }
```

Что тренируем:

1. Объявление двух переменных.
2. Ввод нескольких значений через `Scan`.
3. Использование адреса переменной через символ `&`.
4. Простая арифметика и вывод результата.

Пример:

- Ввод: `3 7`
- Вывод: `10`

## Этап 3: Перевод угла в время на часах

Активный код в файле:

```go
func main() {
	var a int
	fmt.Scan(&a)
	c := a / 30
	d := a % 30 * 2
	fmt.Println("На часах:", c, "часов и ", d, "минут")
}
```

### Логика вычисления

Используются правила:

1. `30` градусов соответствуют `1` часу.
2. `1` градус соответствует `2` минутам.

Формулы:

1. `c = a / 30` - полные часы через целочисленное деление.
2. `d = (a % 30) * 2` - остаток градусов переводим в минуты.

Примеры:

| Вход `a` | Результат |
|---|---|
| `0` | `0 часов 0 минут` |
| `75` | `2 часа 30 минут` |
| `180` | `6 часов 0 минут` |
| `359` | `11 часов 58 минут` |

## Что важно понять по математике

1. Деление целых чисел отбрасывает дробную часть.
2. Остаток от деления показывает, сколько градусов осталось после полных часов.
3. Умножение остатка на `2` переводит градусы в минуты.

## Частые ошибки новичка

1. Забыть символ `&` в `Scan`.
2. Перепутать операции `/` и `%`.
3. Проверять программу только на одном входном значении.
4. Не учитывать, что в файле должен быть только один активный `main`.

3. Для задачи с часами проверь минимум `10` тестов.
